# Brain Tumor Segmentation using U-Net

## Overview
An AI-powered brain tumor segmentation system built using U-Net and Streamlit.

## Features
- MRI image upload
- Brain tumor segmentation
- Interactive Streamlit interface
- Database integration

## Tech Stack
- Python
- TensorFlow/Keras
- Streamlit
- SQLite

## Project Structure
app/
model/
utils/
data/

## Installation

pip install -r requirements.txt

## Run

streamlit run app/app.py

## Model
U-Net model: model/unet_model.h5
